###Простейшее вэб-приложение на Flask
* **Запуск:**
    * Запуск с портом по умолчанию: ```python -m app```
    * Запуск с указанным портом 8001: ```python -m app 8001```
    * Запуск с портом, указанным в переменнной окружения:
         ```
        SET PORT=8001
        python -m app
        ```
  
* **Запуск тестов:**
    ```
    python -m tests
    ```
